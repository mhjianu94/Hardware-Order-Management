Aplicația de Gestionare a Comenzilor de Hardware

Introducere
Această aplicație vă ajută să gestionați comenzi de hardware, urmărind componente individuale, computere și compatibilități 
între componente.

Cum să începeți
După ce ați deschis aplicația, veți vedea o listă de componente, computere și compatibilități. Puteți adăuga, actualiza 
sau șterge orice element din aceste liste.

Componente
Fiecare componentă este un element de hardware individual, cum ar fi un procesor sau o placă de bază. Puteți 
adăuga o nouă componentă prin apăsarea butonului "Adăugați Componentă", completând detaliile necesare și apăsând "OK". 
Pentru a actualiza o componentă existentă, selectați-o din listă și apăsați "Actualizați Componentă". 
Puteți șterge o componentă selectând-o și apăsând "Ștergeți Componentă".

Computere
Un computer este un set de componente. Puteți adăuga un nou computer prin apăsarea butonului 
"Adăugați Computer", completând detaliile necesare și apăsând "OK". Pentru a actualiza un computer existent, 
selectați-l din listă și apăsați "Actualizați Computer". Puteți șterge un computer selectându-l și apăsând "Ștergeți Computer".

Compatibilități
O compatibilitate indică dacă două componente pot funcționa împreună în același computer. Puteți 
adăuga o nouă compatibilitate prin apăsarea butonului "Adăugați Compatibilitate", selectând două componente. 
Pentru a actualiza o compatibilitate existentă, selectați-o din listă și  apăsați "Actualizați Compatibilitate". 
Puteți șterge o compatibilitate selectând-o și apăsând "Ștergeți Compatibilitate".

Detalii Computer
Pentru a vedea detaliile unui computer, faceți dublu-click pe acesta în lista de computere. Acest lucru va deschide 
o fereastră nouă care afișează toate componentele computerului. Puteți adăuga componente la computer selectând din 
meniul derulant și apăsând "Adăugați Componentă". Puteți șterge componente din computer selectând-o și apăsând "Ștergeți 
Componentă".

Shortcut-uri
Ctrl + S: Salvează toate datele curente.
Ctrl + O: Încarcă toate datele salvate anterior.
Ctrl + R: Generează un raport cu toate detaliile actuale.

Generarea de Rapoarte

Puteți genera un raport detaliat al tuturor componentelor, computerelor și compatibilităților apăsând "Generare Raport" 
în meniul principal. Acest raport va conține detalii despre fiecare element și poate fi salvat pentru viitoare referințe.

Salvare și Încărcare
Puteți salva toate datele curente prin apăsarea "Ctrl + S" sau selectând "Salvare" din meniul principal. 
Datele sunt salvate într-un fișier de serializare și pot fi încărcate ulterior.

Pentru a încărca date salvate anterior, apăsați "Ctrl + O" sau selectați "Încărcare" din meniul principal. 
Trebuie să selectați fișierul de serializare pe care doriți să îl încărcați.

Atenție: Încărcarea unui fișier de serializare va înlocui toate datele curente. 
Asigurați-vă că ați salvat orice date pe care doriți să le păstrați înainte de a încărca un fișier de serializare.

In folderul Data puteti gasi exporturi si salvari.
